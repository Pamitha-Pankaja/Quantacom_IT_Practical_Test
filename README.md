# Employee Management System

This project is an Employee Management System implemented using Spring Boot, JPA, and JasperReports. It includes user authentication and authorization with JWT, CRUD operations for employees, profile picture upload and download functionality, scheduled tasks for updating employee age, and PDF report generation.

## Features

- User authentication and authorization with JWT
- CRUD operations for employees
- Profile picture upload and download
- Scheduled tasks for updating employee age
- PDF report generation with JasperReports
- Role-based access control for sensitive operations
